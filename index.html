Práctica 7: Casa con Flores en AutoCAD

Vista Previa del Resultado Final

1. Configuración Inicial

  1. Abre AutoCAD y crea un nuevo dibujo.
  2. Configura las unidades: UNIDADES → Decimal, precisión 0.00.
  3. Activa ORTO (F8) y SNAP (F9).
  4. Establece los límites del dibujo: LIMITS → 0,0 → 400,300.

2. Dibujar la Casa

Base de la Casa

Comando: REC → Enter

Primera esquina: 100,150 → Enter

Esquina opuesta: @200,120 → Enter

Techo

Comando: L → Enter

Puntos: 100,270 → 200,350 → 300,270 → C (para cerrar)

3. Agregar Puerta y Ventanas

Puerta

Comando: REC → Enter

Primera esquina: 170,150 → Enter

Esquina opuesta: @60,50 → Enter

Ventana Izquierda

Comando: REC → Enter

Primera esquina: 120,200 → Enter

Esquina opuesta: @40,40 → Enter

Ventana Derecha

Comando: REC → Enter

Primera esquina: 240,200 → Enter

Esquina opuesta: @40,40 → Enter

4. Dibujar Flores

Flor Izquierda

1. Dibujar el tallo:

Comando: L → Enter

Puntos: 80,120 → 80,170 → Enter

2. Dibujar los pétalos:

Comando: C → Enter

Centro: 80,170 → Enter

Radio: 15 → Enter

3. Dibujar el centro de la flor:

Comando: C → Enter

Centro: 80,170 → Enter

Radio: 5 → Enter

Flor Derecha

1. Dibujar el tallo:

Comando: L → Enter

Puntos: 300,120 → 80,170 → Enter

2. Dibujar los pétalos:

Comando: C → Enter

Centro: 300,170 → Enter

Radio: 15 → Enter

3. Dibujar el centro de la flor:

Comando: C → Enter

Centro: 300,170 → Enter

Radio: 5 → Enter

5. Agregar el Sol

Comando: C → Enter

Centro: 330,350 → Enter

Radio: 25 → Enter

6. Colorear el Dibujo

1. Seleccionar la herramienta SOMBREA (SOMBREADO)

2. Seleccionar los siguientes elementos y aplicar colores:

  • Casa: Amarillo claro (#f9e4b7)
  • Techo: Rojo (#d9534f)
  • Puerta: Marrón (#8b4513)
  • Ventanas: Azul claro (#add8e6)
  • Flores: Rosas con centro amarillo (#ff69b4 y #ffff00)
  • Tallos: Verde (#228b22)
  • Sol: Amarillo (#ffd700)

Consejos:

  • Usa ZOOM para acercar o alejar la vista según sea necesario.
  • Guarda tu trabajo frecuentemente con CTRL + S.
